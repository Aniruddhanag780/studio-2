# **App Name**: Lyra AI Console

## Core Features:

- AI Agent Overview & Marketing: Display the core messaging, supported AI models (Gemini, GPT, Claude), and a prominent 'Get Started' call to action, replicating the initial landing page section.
- Dynamic AI System Status Dashboard: Present a grid of customizable cards showing the status and simulated output for various AI agents and system components like 'TEST-GEN', 'DEPLOY', 'EVENT LOG', and 'TASK QUEUE'.
- Simulated Agent Output Generation Tool: Utilize generative AI as a tool to create realistic, context-specific command-line output, log entries, and status messages for each dashboard card, making the simulated environment feel dynamic.
- Interactive Status Indicators: Implement visually distinctive and possibly animated status indicators (e.g., green/red dots) next to each agent card name to reflect operational state.
- Basic Application Navigation: Provide a top-level navigation bar with links for 'Platform', 'Pricing', 'Docs', 'Login', and 'Get Access' to facilitate basic site structure and user interaction.

## Style Guidelines:

- Primary color: Vibrant golden yellow (#FFDD33) used for key textual highlights and interactive elements, conveying energy and digital luminescence against a dark theme.
- Background color: Deep charcoal with a subtle warm undertone (#0F0E0B), creating a sophisticated, high-contrast canvas that accentuates other UI elements and text.
- Accent color: Bold reddish-orange (#F23F0D) for strong calls to action, important alerts, and to provide a secondary layer of visual emphasis.
- Headline font: 'Space Grotesk' (sans-serif) for its computerized and modern feel, perfect for prominent titles like AI model names.
- Body font: 'Inter' (sans-serif) for general UI text, navigation, and descriptions, chosen for its clear and objective readability.
- Code/Log font: 'Source Code Pro' (monospace) specifically for content within the agent and system log cards, ensuring legibility of code-like output.
- Use minimalist, outline-style icons with high contrast, aligning with the clean, modern aesthetic of a console interface. Status indicators should be simple dots.
- A spacious, two-column layout: a prominent hero section on the left and a responsive, grid-based dashboard for agent/system cards on the right. Dark theme with high contrast between text and background.
- Subtle and fluid animations for hover states, active status indicators, and seamless transitions during any dynamic content updates within the dashboard cards.
